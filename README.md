# Sales Dashboard Intelligence

A premium Next.js 15 application built on Atomic Design principles to visualize sales operations historically across multiple years (2022-2024). It dynamically filters numerical limits, visualizes data in multiple forms (Bar, Line, Pie), and provides robust market intelligence tracking.

## 🚀 Features

- **Atomic Design System:** Strictly organized by `Atoms`, `Molecules`, and `Organisms` to structure UI components intuitively and scalably.
- **Multiple Charts:** Visualize numbers based on Recharts components representing Bar, Line, or Pie structures interactively based on user inputs.
- **Micro-Filters:** Threshold capabilities to analyze top earners via custom threshold filters above basic averages, built from scratch using `lucide-react`.
- **API Simulation Model:** Contains a `Next.js Route Handler` to mimic the asynchronous fetching of real external API data.
- **Dark-Theme Aesthetics:** A vibrant, dark-themed responsive UI showcasing the best of utility-first CSS via `Tailwind CSS 4.0`.

## 🛠 Tech Stack

- Next.js 15 (App Router, Route Handlers)
- TypeScript
- Tailwind CSS 4.0
- Recharts
- Lucide React

## 📦 Local Setup Procedures

**1. Clone the repository:**
```bash
git clone <your-repository-url>
cd sales-dashboard
```

**2. Install required dependencies:**
```bash
npm install
```

**3. Fire up the development environment:**
```bash
npm run dev
```

Your localized application will now be running correctly at [http://localhost:3000](http://localhost:3000). Enjoy intelligent analytics at speed with Next.js 15!
